# JD_crawl_spider
京东众筹爬虫，导出Excel

可以改变categoryId参数改变爬取数据分类

每次爬取需要移除文件夹中的Excel文件，或改变源代码中的Excel表命名

## 参数说明

参数|参数说明
:---:|:---:
name|名称
process|进度
realmoney|筹集金额
aimmoney|目标金额
startdate|开始时间
enddate|结束时间
totalpic|图片数目
totalvideo|视频数目
reportcount|无偿支持者
prepost|项目发起人发起项目数
presupport|项目发起人支持项目数
support|项目支持持人数
ountprice|所有价位
totalprice|价位数
totalscore|项目评分
picurl|项目图片地址
